# AX-On Platform 개요 및 메뉴 구성

> **문서 버전**: v3.0
> **작성일**: 2026-03-03
> **브랜드명**: AX-On Platform (AX 온 플랫폼)
> **도메인**: www.ax-on.net

---

## 1. 플랫폼 개요

### 1.1 명칭
**AX-On Platform** (AX 온 플랫폼)

### 1.2 네이밍 의미
- **AX** = AI Transformation (AI 전환)
- **On** = "켜다", "시작하다", "플랫폼 위에서"
- **AX-On** → "액션(Action)"과 유사한 발음 → 실행력·행동의 뉘앙스
- 즉, **"AI 전환을 켜다" + "액션하다"**

### 1.3 플랫폼 목적
분야별 AI 전문가들이 시니어 AI 창업 교육과 기업 AX 프로젝트를 추진하고, 모든 참여자가 커뮤니티에서 만나는 AI 활용 플랫폼

---

## 2. 메뉴 구성 (4대 메뉴 + 플로팅 챗봇)

| # | 메뉴 | 내용 |
|---|------|------|
| 1 | **우리의 창업** (시니어 AI 창업 교육) | AI 전문가가 강사로 참여하는 시니어 대상 AI 활용 창업 교육 프로그램 |
| 2 | **AX 프로젝트** | 기업 AX 프로젝트 접수, 분야별 AI 전문가들이 직접 프로젝트 추진 |
| 3 | **AI 전문가** | 분야별 AI 전문가 등록·관리 |
| 4 | **커뮤니티** | 모든 참여자가 교류·활동하는 공간 (하위 메뉴: AI 활용법) |
| 🤖 | **AI 튜터** (플로팅 챗봇) | 지식베이스 기반 대화형 질의응답 — 모든 페이지에서 접근 가능 |

### 네비게이션 구성
- **좌측**: 우리의 창업 | AX 프로젝트 | AI 전문가 | 커뮤니티
- **우측**: 로그인 | 회원가입

> **참고**: AI 활용법은 독립 메뉴가 아닌 커뮤니티의 하위 메뉴로 배치. 커뮤니티 페이지 내에서 접근 가능.

## 3. 각 메뉴 상세

### 3.1 우리의 창업 — 시니어 AI 창업 교육
- **브랜드명**: 우리의 창업
- **대상**: AI를 활용하여 창업하려는 시니어
- **형태**: 교육 프로그램 운영
- **강사**: 분야별 AI 전문가가 강사로 직접 참여
- **목표**: 시니어들이 AI 활용 역량을 갖추고 창업할 수 있도록 지원

### 3.2 AX 프로젝트
- **대상 기업**: 소기업, 중기업, 중견기업
- **기능**: 기업의 AX 프로젝트 접수, 분야별 AI 전문가들이 직접 프로젝트 추진
- **규모**: 155+ AX 모듈, 15개 기능 분야, 14개 산업 분류, 5단계 AX 로드맵
- **목표**: 기업의 전사적 AX(AI Transformation) 전환 프로젝트 추진

### 3.3 AI 전문가
- **대상**: 분야별 AI 전문가
- **기능**: 전문가 등록, 프로필 관리, 분야·역량별 검색
- **역할**:
  - 시니어 AI 창업 교육에 **강사로 참여**
  - 기업 AX 프로젝트를 **직접 수행**
- **목표**: 교육과 프로젝트 양쪽에서 활동하는 AI 전문가 풀 구성·관리

### 3.4 커뮤니티
- **참여자 4그룹**:
  - 시니어 AI 창업자
  - AI 전문가
  - AX 추진 기업 (소기업·중기업·중견기업)
  - AI에 관심 있는 일반 참여자
- **하위 메뉴**: AI 활용법
- **목표**: 모든 참여자가 경험과 노하우를 공유하는 교류의 장

### 3.4.1 AI 활용법 (커뮤니티 하위)
- **내용**: AI 활용법 지식을 체계적으로 축적한 지식베이스
- **운영 방식**:
  - GitHub 기반 위키 방식 (위키피디아처럼 누구나 기여 가능)
  - SAL Grid (Stage × Area × Level) 3D 좌표계 구조로 체계화
  - 마크다운 문서 → HTML 변환 → 웹사이트로 탐색
- **접근 방식**: 커뮤니티 페이지 내 메뉴로 접근, HTML 웹페이지를 통한 직접 탐색·검색

### 🤖 AI 튜터 (플로팅 챗봇)
- **형태**: 플로팅 챗봇 — 모든 페이지 우측 하단에 상시 노출
- **기능**: AI 활용법(지식베이스) 기반 대화형 질의응답
- **동작**: 사용자가 "이거 어떻게 해?" → 챗봇이 KB에서 찾아 답변
- **기술**: 지식베이스 RAG(Retrieval Augmented Generation) 기반

## 4. 선순환 구조

AI 전문가를 중심으로 교육과 프로젝트 양쪽으로 분기되어 활동하고, 모든 참여자가 커뮤니티에서 다시 합류하는 구조.

```
              ┌─────────────────┐
              │  AI 전문가 풀    │  ← 중심
              │ (전문가 등록·관리)│
              └──────┬──────────┘
                     │
           ┌─────────┴─────────┐
           ↓                   ↓
  ┌────────────────┐  ┌────────────────┐
  │ 시니어 AI      │  │ AX 프로젝트    │
  │ 창업 교육      │  │ (기업 AX 추진) │
  │ (강사로 참여)  │  │ (직접 수행)    │
  └───────┬────────┘  └───────┬────────┘
          │                   │
          └─────────┬─────────┘
                    ↓
           ┌────────────────┐
           │   커뮤니티      │  ← 합류
           │ (모든 참여자    │
           │  교류·공유)     │
           └───────┬────────┘
                   │
                   ↻ 다시 전문가 역량 강화로 순환
```
